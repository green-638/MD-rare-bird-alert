Help

Usage of this site requires an eBird API key. To generate a key, an eBird account is required. Click the link below generate a key.

Generate key

Site functions

Rare bird alert

Use this page to create alerts. An API key must be entered before searching for locations. At least one county or hotspot must be selected. Once locations are selected and the form is submitted, an alert will be posted to database for each location. API keys are not stored in the database. You will receieve an email if there are rare birds reported during your indicated interval. Emails are currently sent around midnight.

Rare bird data

Use this page to search for reports of rare birds during the last 30 days. An API key must be entered before searching for locations. At least one county or hotspot must be selected. You then must enter a timeframe to search for data. Duplicate reports (same species, location, and day) will be removed, and only the latest duplicate report will be shown.

Manage alerts

Use this page to see your alerts and delete them. Enter an email to see the alerts assigned to it.
